\doxysubsection{Detailed Description}
The \doxylink{classdi_1_1uniba_1_1map_1_1b_1_1adventure_1_1_engine}{Engine} class is the main driver of the game. It initializes the game environment, including the game description, parser, and window. It also handles the execution of the game by displaying the start description, room name, and room description.
